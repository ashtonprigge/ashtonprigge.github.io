Mobirise

The main difference between the document portfolio and the website portfolio is that the pdf version is a fixed document, while my website will be updated as I progress through my career. The functionality of the two are fundmentally different, though both convey the same information. The print version is a single document while the website requires navigation through seperate pages. 

I have provided a pdf version of the portfolio as it is a common format and does not require a purchase.I created my website using the application Mobirise which provides a simple interface for creation of a website as well as the ability to download and edit the website code. I am hosting my website on github pages due to its longevity and lack of cost.

My font and color scheme were selected for easy readability. The headings and naming of my pages are accurate reflections of the content. I also included captions with my photographs. These design choices were made to aid any user utilizing a screen reader.
